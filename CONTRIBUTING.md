Contributing to AthenaPass

Thanks for your interest in contributing! This file explains how to contribute
and lists current contributors.

Contributors
------------

- RyoSh1 <p.juncal.moreira@udc.es>

If you contributed and your name is missing or you prefer a different
attribution, open an issue or send a PR with the corrected name/email.

How to contribute
-----------------
- Fork the repository and create a branch for your change.
- Write clear commit messages and keep changes focused.
- Run existing tests and linters (if any) before opening a PR.
- Open a Pull Request with a description of the change and mention any
  relevant issues.

Code of Conduct
---------------
Be respectful and constructive. Treat others as you expect to be treated.

Reporting Security Issues
-------------------------
If you discover a security vulnerability, please disclose it privately by
opening an issue and marking it as "security" or contact the repository
maintainers directly at ruben.diz@udc.es or p.juncal.moreira@udc.es.
